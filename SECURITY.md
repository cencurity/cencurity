# Security Policy

## Supported Versions

This repository contains a deploy-only package.
Support is provided on a best-effort basis for the latest `main` branch.

## Reporting a Vulnerability

If you believe you’ve found a security issue, please do **not** open a public issue.

- Email: `security@centramcp.com` (preferred)
- If you cannot email, open a private GitHub Security Advisory (if enabled).

Please include:
- A clear description of impact
- Reproduction steps or a PoC (if available)
- Affected version/tag (if known)

We’ll acknowledge receipt and follow up as soon as possible.
