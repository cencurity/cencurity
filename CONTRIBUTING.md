# Contributing

Thanks for your interest in improving Cencurity Community.

## What you can help with

- Documentation improvements (README, troubleshooting, setup)
- Security hardening suggestions for the deploy package
- Bug reports with clear repro steps

## How to contribute

1. Fork the repo
2. Create a feature branch
3. Keep changes focused and minimal
4. Open a Pull Request describing the motivation and expected behavior

## Reporting issues

When filing an issue, include:
- OS + Docker version
- `docker compose ps` output
- Relevant container logs (`docker logs <container>`)
- Steps to reproduce

## Security issues

Please follow [SECURITY.md](SECURITY.md) instead of opening a public issue.
